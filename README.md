# Software_Engineering_12_13

ソフトウェア工学の課題12-13用リポジトリ。

## 概要

このリポジトリでは、Gitによるバージョン管理とGitHubを用いた開発フローを確認するために、Pythonプログラムを管理する。

対象のプログラム `main.py` では、広さ・築年数・価格のデータを用いて重回帰分析を行う。  
説明変数として広さと築年数、目的変数として価格を用い、回帰係数、分散比、決定係数、自由度調整済み決定係数、標準化残差、テコ比、信頼区間、予測区間を計算する。（感性情報処理）

## ファイル構成

- main.py：重回帰分析を行うPythonプログラム
- README.md：リポジトリの説明ファイル
- .gitignore：Gitで管理しないファイルを指定する設定ファイル

以下のコマンドで実行する。
